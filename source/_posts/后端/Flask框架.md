Flask是一个简单的Python后端框架，用于构建简单的Python后端Web应用程序。它提供了基本的路由、请求处理和响应机制，使开发人员能够专注于业务逻辑而不是底层实现。

<!--more-->

## Flask安装

```bash
conda install flask
```

## Flask教程

有关Flask的详细内容可以参考[Flask的官方文档](https://flask.org.cn/en/stable/)。
