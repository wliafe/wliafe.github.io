Ollama是一个基于LLM的模型服务，支持多种模型，包括但不限于Llama、Llama-2、Llama-2-7b-chat、Llama-2-13b-chat、Llama-2-70b-chat等。

<!--more-->

## Ollama使用命令

显示模型列表。

```bash
ollama list
```

显示模型的信息

```bash
ollama show
```

拉取模型

```bash
ollama pull
```

推送模型

```bash
ollama push
```

拷贝一个模型

```bash
ollama cp
```

删除一个模型

```bash
ollama rm
```

运行一个模型

```bash
ollama run
```
